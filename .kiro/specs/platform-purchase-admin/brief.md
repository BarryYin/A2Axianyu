# Brief: 平台代购管理员页面

## Problem
AI代理可以自动议价成交，但闲鱼的实际下单需要人工操作（支付认证、反自动化策略）。需要一个高效的管理员界面来桥接AI成交与真实采购。

## Current State
- A2A闲鱼后端API和数据库schema已基本完成
- 前端UI组件为空
- 管理员页面完全未开发
- 无订单状态管理系统

## Desired Outcome
管理员能在一个页面上看到所有待采购订单，一键跳转闲鱼链接完成采购，然后标记结果。系统自动通知买家。

## Approach
Human-in-the-loop: AI负责谈判，人负责执行闲鱼采购，AI负责通知买家。

## Scope
- In: 管理员订单面板、订单状态管理、买家通知、产品上架、AI议价规则
- Out: 闲鱼自动化下单、支付处理、卖家端功能、物流对接

## Core Flow
```
AI议价成交 → 买家付款 → 生成待办 → 人工闲鱼采购 → 标记结果 → AI通知买家
```

## Key Metrics
- 管理员处理每单时间 < 2分钟
- 买家通知送达率 > 95%
- 代购成功率 > 60%
- 退款率 < 40%
